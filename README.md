## 环境与依赖
* Python 2.7
* Requests包
## 使用说明
文件目录下创建config.py，键入：
```Python
chapter="" # 课程章节编号
cookie="" # 登录后缓存的Cookie
user_agent="" # 浏览器UA，伪装用，聊胜于无
```
例如：
```Python
chapter="19537"
cookie="key1=value1;KEY2=VAULE2"
user_agent="Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:57.0) Gecko/20100101 Firefox/57.0"
```
之后解释运行main.py，文件目录下执行python main.py即可
